Operations

extrude

(extrude shape &keys :h :x :y :z :dir :both :eager :hide)

Extrude a shape or wire along a direction. Keywords: :h (height), :x/:y/:z (axis shortcuts), :dir [dx dy dz] (direction vector), :both (extrude both sides), :eager, :hide

(extrude wire :h 10)
(extrude wire :h 10 :x)
(extrude wire :h 10 :dir [0 0 1] :both)
Returns a rojcad/shape abstract value.

extrude-polygon

(extrude-polygon points height &keys :plane :at :eager :hide)

Extrude a polygon defined by a list of 2D points. Keywords: :plane (keyword, default :xy), :at [x y z], :eager, :hide

(extrude-polygon [[0 0] [10 0] [10 10] [0 10]] 5)
(extrude-polygon [[0 0] [10 0] [5 10]] 5 :eager)
Returns a rojcad/shape abstract value.

revolve

(revolve shape &keys :a :ar :c :dir :eager :hide)

Revolve a shape around an axis to create a solid. Keywords: :a (angle in degrees), :ar (angle in radians), :c [x y z] (axis origin), :dir [dx dy dz] (axis direction), :eager, :hide

(revolve shape :a 90)
(revolve shape :ar math/pi)
(revolve shape :c [0 0 0] :dir [0 1 0] :a 180)
Returns a rojcad/shape abstract value.

Selection

on-select

(on-select callback)

Register a function to be called on selection events. Pass nil to unregister.

(on-select (fn [s] (print "selected: " s)))
(on-select nil)  # unregister
Returns nil.

poll-selection

(poll-selection)

This is called internally by the event loop.

(poll-selection)  # returns shape, keyword, tuple, or nil
Poll for a selection event. Returns a shape if one is selected, :deselected if all deselected, [:deselected id] if a specific shape was deselected, or nil if no event.

I/O

read-step

(read-step path &keys :eager :hide)

Read a STEP file from disk and return a shape.

(read-step "/tmp/model.step")       -- load from file
(read-step "/tmp/model.step" :eager) -- load and tessellate
(read-step "/tmp/model.step" :hide)  -- load without showing
Returns a rojcad/shape abstract value. Signals an error on failure.

write-step

(write-step shape path)
(write-step my-shape "/tmp/model.step")
Export a shape to a STEP file at the given path. Returns nil on success, signals an error on failure.

write-stl

(write-stl shape path)
(write-stl my-shape "/tmp/model.stl")
Export a shape to an STL file at the given path. Returns nil on success, signals an error on failure.

(view-angle yaw pitch &opt distance)

Set the 3D viewport camera angle.

yaw (radians), pitch (radians), distance (optional, default 100).

view-back

(view-view-back ; distance)

Set camera to back view (looking along -Z toward origin). Yaw=-π/2, Pitch=0. Animates over 0.5s. Optional distance sets zoom level; omitted preserves current.

(view-back)
(view-back 200)

view-bottom

(view-view-bottom ; distance)

Set camera to bottom view (looking along -Y toward origin). Yaw=0, Pitch=-π/2. Animates over 0.5s. Optional distance sets zoom level; omitted preserves current.

(view-view-iso ; distance)

Set camera to isometric view (looking from (1,1,1) direction). Yaw=π/4, Pitch=asin(1/√3) ≈ 0.615 rad. Animates over 0.5s. Optional distance sets zoom level; omitted preserves current.

(view-iso)
(view-iso 150)

Primitives

box

(box &keys :w :d :h :c :pl :ph :eager :hide)

Create a box or cube.

Positional: (box w d h) or (box size) for a cube. Keywords: :w :d :h (dimensions), :c (center [x y z]), :pl :ph (opposite corners [x y z]). :eager (tessellate immediately). :hide (skip automatic show on def).

(box 10 20 30)           # box at origin
(box 10 20 30 :c [5 5 5]) # centered box
(box 5)                  # 5x5x5 cube
(box :pl [0 0 0] :ph [10 20 30]) # from corners
(box :w 10 :d 20 :h 30) # keyword style
(box 10 :eager)          # eager tessellation
(box 10 :hide)           # create without showing
Returns a rojcad/shape abstract value.

cone

(cone &keys :br :tr :h :c :a :ar :eager :hide)

Create a cone or truncated cone. Keywords: :br (base radius), :tr (top radius, default 0), :h (height), :c (center [x y z]), :a (angle in degrees), :ar (angle in radians), :eager, :hide

(cone 5 10)                   # cone radius 5 height 10
(cone 5 2 10)                 # truncated cone
(cone :br 5 :tr 2 :h 10)      # keyword style
(cone 5 10 :eager)            # eager tessellation
Returns a rojcad/shape abstract value.

cylinder

(cylinder &keys :r :h :c :dir :fp :tp :eager :hide)

Create a cylinder.

Positional: (cylinder radius height) - along Z axis, base at Z=0 Keywords: :r (radius), :h (height), :c (center [x y z]), :dir (direction [dx dy dz]), :fp (from-point [x y z]), :tp (to-point [x y z]). :eager (tessellate immediately).

(cylinder 5 10)                       # simple
(cylinder 5 10 :c [0 0 5])            # centered
(cylinder :fp [0 0 0] :tp [0 0 10] :r 5) # point-to-point
(cylinder :r 5 :h 10)                 # keyword style
(cylinder 5 10 :eager)                # eager tessellation
Returns a rojcad/shape abstract value.

sphere

(sphere &keys :r :c :a :ar :eager :hide)

Create a sphere. Keywords: :r (radius), :c (center [x y z]), :a (angle in degrees), :ar (angle in radians), :eager, :hide

(sphere 5)                    # radius 5 at origin
(sphere :r 5 :c [1 2 3])      # centered
(sphere 5 :a 90)              # hemisphere
(sphere 5 :eager)             # eager tessellation
Returns a rojcad/shape abstract value.

torus

(torus &keys :rr :tr :c :a :ar :as :asr :ae :aer :dir :eager :hide)

Create a torus.

Positional: (torus rr tr) Keywords: :rr (ring radius), :tr (tube radius), :c (center [x y z]), :a (angle in degrees), :ar (angle in radians, partial), :as (start angle degrees), :asr (start angle radians), :ae (end angle degrees), :aer (end angle radians), :dir (axis direction [dx dy dz]), :eager (tessellate immediately).

(torus 20 10)                    # full torus
(torus 20 10 :c [0 0 5])         # repositioned
(torus 20 10 :a 180)             # half torus
(torus :rr 20 :tr 10 :as 0 :ae 180) # angled range
(torus :rr 20 :tr 10 :dir [0 1 0]) # oriented
(torus 20 10 :eager)             # eager tessellation
Returns a rojcad/shape abstract value.

Sketch

arc-to

(arc-to sketch x2 y2 x3 y3)
Draw a circular arc through (x2, y2) to (x3, y3). Returns a new sketch.

build-wire

(build-wire sketch &keys :eager :hide)

Return the sketch as an unclosed Wire. Keywords: :eager, :hide

(-&gt; (sketch) (line-to 10 0) (line-to 10 10) (build-wire))

close-sketch

(close-sketch sketch &keys :eager :hide)

Close the sketch and return a Face. Keywords: :eager, :hide

(-&gt; (sketch) (line-to 10 0) (line-to 10 10) (close-sketch))

line-dx

(line-dx sketch dx)
Draw a horizontal line by dx units. Returns a new sketch.

line-dx-dy

(line-dx-dy sketch dx dy)
Draw a line by (dx, dy) offset. Returns a new sketch.

line-dy

(line-dy sketch dy)
Draw a vertical line by dy units. Returns a new sketch.

line-to

(line-to sketch x y)
Draw a line from current cursor to (x, y). Returns a new sketch.

move-to

(move-to sketch x y)
Move the sketch cursor to (x, y) without drawing. Returns a new sketch.

sketch

Create a new sketch on a workplane. Keywords: :plane (keyword, default :xy), :at (array [x y z]). Returns a rojcad/sketch abstract value.

2D Primitives

circle

(circle &keys :r :wire :plane :at :eager :hide)

Create a circle. Keywords: :r (radius), :wire (output as wire), :plane (keyword, default :xy), :at [x y z], :eager, :hide

(circle 5)
(circle :r 5 :plane :xz)
(circle 5 :wire)
Returns a rojcad/shape abstract value.

polygon

(polygon &keys :pts :wire :plane :at :eager :hide)

Create a polygon from a list of 2D points. Keywords: :pts (array of [x y] points), :wire (output as wire), :plane (keyword, default :xy), :at [x y z], :eager, :hide

(polygon :pts [[0 0] [10 0] [5 10]])
(polygon :pts [[0 0] [10 0] [10 10] [0 10]] :wire)
Returns a rojcad/shape abstract value.

rect

(rect &keys :w :d :h :wire :plane :at :eager :hide)

Create a rectangle. Keywords: :w (width), :d/:h (depth), :wire (output as wire), :plane (keyword, default :xy), :at [x y z], :eager, :hide

(rect 10 20)

Text

list-fonts

(list-fonts)
(list-fonts)  # returns @[["Arial" "/path/Arial.ttf" :regular] ...]
List available system fonts. Returns an array of [name type style] tuples, where type is the font file path and style is a keyword.

text

(text str font size &keys :depth :both :plane :at :eager :hide)

Create 2D text from a TrueType/OpenType font. :font is a font name string, :size is the font size. Keywords: :depth, :both (extrude both sides), :plane (keyword), :at [x y z], :eager, :hide

(text "Hello" "Arial" 10)
(text "Hello" "Arial" 10 :depth 5)
Returns a rojcad/shape abstract value.

text3d

(text3d str font size depth &keys :both :plane :at :eager :hide)

Create 3D text (extruded) from a TrueType/OpenType font. :font is a font name string, :size is the font size. Keywords: :both (extrude both sides), :plane (keyword), :at [x y z], :eager, :hide

(text3d "Hello" "Arial" 10 5)
Returns a rojcad/shape abstract value.

Registry

hide

(hide & shapes)

Set shapes' visible flag to false. Shapes stay registered but are no longer rendered.

(hide my-shape)
(hide shape-a shape-b)
Returns nil.

purge

(purge & shapes)

Remove shapes from the viewer registry and mark them as purged. They will no longer be rendered. Use (def name nil) to unbind.

(purge my-shape)
Returns nil.

registry-remove

(registry-remove & shapes)

Immediately remove shapes from the viewer registry. Used internally by `purge`. The underlying OCCT shape memory is freed when Janet's GC collects the shape value.

Returns nil.

show

(show & shapes)

Register shapes in the viewer and make them visible. Calling show on an already-visible shape is a no-op.

(show my-shape)
(show shape-a shape-b)
Returns nil.

Transforms

mirror

(mirror shape ox oy oz dx dy dz &keys :eager :hide)

Mirror a shape across a plane defined by a point and normal. :ox oy oz is a point on the plane. :dx dy dz is the plane normal. Keywords: :eager, :hide

rotate

(rotate shape &keys :a :ar :x :y :z :r :eager :hide)

(rotate box :z :a 90)
(rotate box :x :ar math/pi)
(rotate box :r [0 1 0] :a 45)

scale

(scale shape factor &keys :o :eager :hide)

Uniformly scale a shape by a factor. Keywords: :o [x y z] (origin for scaling), :eager, :hide

(scale box 2)                # double size
(scale box 2 :o [5 5 5])     # scale from center
Returns a rojcad/shape abstract value.

translate

Translate (move) a shape by a vector. Positional: (translate shape dx dy dz) Keywords: :t [dx dy dz], :eager, :hide

(translate box :t [10 0 0])
Returns a rojcad/shape abstract value.
