CAD operations — Rust wrappers around opencascade-rs.

Implements: box, sphere, cylinder, cone, torus, cut, common, shape type, export.

Constants§

SYNTHETIC_WIREFRAME_THRESHOLD
Number of topological edges below which we add synthetic wireframe edges. Polyhedral shapes (box has 12 edges) keep only topological edges; curved shapes (sphere has 1 seam edge) get the synthetic fallback.
