rojcad — Headless parametric CAD system with embedded Janet DSL.

This binary embeds the Janet interpreter, registers CAD functions (box, sphere, cylinder, cone, torus, cut, common, shape-type, hide, show, visible?, write-step, write-stl), and starts two TCP REPL servers: a raw-text REPL on port 9364 (configurable via –raw-port) and a spork netrepl protocol server on port 9365 (configurable via –spork-port).
