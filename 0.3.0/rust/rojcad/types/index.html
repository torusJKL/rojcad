Source

Shape data type — a Janet abstract type wrapping an OCCT Shape with metadata for visibility, color, etc.

Also defines the shared ShapeRegistry used to synchronize shape state between the REPL thread and the viewer thread.

Last selected shape ID, used to propagate selection events to Janet. 0 = no event pending, u64::MAX = deselected, other = selected shape ID.

Action type for the last selection event. 0 = none, 1 = toggled_on, 2 = toggled_off, 3 = cleared.

Camera projection mode. true = perspective, false = orthographic. Viewer thread syncs from this each frame.

Set to true when the viewer requests the application to quit (Ctrl+Q or window close).
REGISTRY_GENERATION
Global generation counter for change tracking. Incremented on every ShapeRegistry write. The viewer reads this to detect changes since last frame.
SELECTED_IDS
Global set of currently selected shape IDs, synced from the viewer thread.
