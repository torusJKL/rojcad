Expand description

A 3D shape with associated metadata.

This is the core data type wrapped as a Janet abstract value. It carries the OCCT Shape plus metadata used by the viewer (visible, color).

Shapes are NOT automatically registered in the viewer registry on creation. Registration happens only when show is explicitly called.

pub fn new(shape: Shape) -> Self

Create a new shape with a unique ID.

The shape is NOT registered in the viewer registry and NOT tessellated. Call show to register and optionally :eager at creation to tessellate early.

Tessellate the shape if not already tessellated. Extracts mesh and edge polylines from the OCCT shape.
